# React Assignment - User Form (中原 錦陽)

## 概要
氏名（漢字/カタカナ）、年齢、性別を入力するユーザー情報フォームです。  
必須チェック、バリデーション、エラーメッセージ表示を実装しています。

## 主な機能
- **必須チェック**  
  未入力やエラーがある場合は「次へ」ボタンが無効化されます。
- **バリデーション**
  - 姓・名：数字・記号を禁止
  - 姓（カナ）・名（カナ）：全角カタカナのみ許可
  - 年齢：全角数字を半角に変換して数値チェック
- **UI**
  - 入力中は黄色枠
  - エラー時は赤枠とエラーメッセージ表示
- **送信時**
  - 全項目が正しく入力されると「complete」とアラート表示
